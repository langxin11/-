## ⚠️ 极其重要：开始前的准备

1. **编译器设置**：
   - 点击左上角 **Menu**。
   - 确保 **Compiler** 选的是 **XeLaTeX** (不要选 pdfLaTeX，否则中文会乱码)。
   
2. **字体文件**：
   - 确保左侧文件列表中有一个 `fonts` 文件夹，里面有 `simsun.ttc` (宋体), `times.ttf` (新罗马) 等文件。如果报错说找不到字体，请联系组长。

## ��️ 如何开始你的任务

我已经把整个论文的框架搭好了。大家不需要动排版代码，只需要**填空**。

1. **找到你的位置**：
   - 打开 `main.tex`。
   - 使用 `Ctrl + F` 搜索红色的 `[TODO]` 标记。
   - 我已经按分工标注好了，例如 `\todo{【第一组成员】...}`。

2. **开始写作**：
   - 将 `\todo{...}` 这一行代码删掉，替换为你翻译好的中文内容。
   - **换行**：在 LaTeX 中，**空一行**才代表换段落。
     ```latex
     这是第一段。
     (这里必须空一行)
     这是第二段。
     ```

## �� 常用功能速查表 (复制粘贴即可)

### 1. 插入数学公式 (Math)
不要直接打 x, y, z，要用数学模式：
- **行内公式** (夹在文字里)：使用 `$` 符号。
  - 写法：`推力 $F_T$ 与力矩 $M_i$ 成正比。`
  - 效果：推力 $F_T$ 与力矩 $M_i$ 成正比。
- **独占一行公式**：
  ```latex
  \begin{equation}
      x_{k+1} = Ax_k + Bu_k
  \end{equation}

### 2. 插入图片 (Images)

1. 点击左上角上传按钮 (Upload)，把图片传上去。
2. 在你要插图的地方复制这段代码：


```latex
\begin{figure}[H] % [H] 表示强制图片在这里，不乱跑
    \centering
    \includegraphics[width=0.8\textwidth]{你的图片文件名.png} % 修改文件名
    \caption{这里写中文图注} % 修改图注
    \label{fig:example} % 给图片起个代号，方便引用
\end{figure}
```



### 3. 引用参考文献 (Citations)

不要手写 `[1]`！

1. 打开左侧的 `ref.bib` 文件，把谷歌学术/百度学术导出的 BibTeX 格式粘贴进去。
2. 记住第一行的那个代号（比如 `nguyen2021model`）。
3. 在 `main.tex` 正文中想引用的地方写：
```latex
正如文献 \cite{nguyen2021model} 所述...

```


它会自动变成 `正如文献 [1] 所述...`

## �� 常见避坑指南

1. **中文显示方框/乱码？**
* 检查是不是 Menu 里编译器没选 **XeLaTeX**。


2. **编译报错了？**
* 不要慌。通常是因为少了半个大括号 `}` 或者公式里的特殊字符（如 `%`, `_`, `&`）没加转义符 `\`。
* 点击 Recompile 按钮旁边的 Logs 图标查看错误行数。


3. **公式太长超出边界？**
* 暂时不管，先把内容翻完，最后统一调整。
